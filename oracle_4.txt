.와일드 카드

: Like 연산자는 검색하려는 값을 정확히 모를 경우에도 검색할수 있도록
와일드 카드와 함께 사용하여 원하는 내용을 검색 하도록 합니다.


와일드 카드				의미
   
     %				문자가 없거나, 하나 이상의 문자에 어떤 문자가 와도 상관 없습니다.

     _				하나의 문자에 어떤 값이 와도 상관없습니다.


	: 사원이름에 마지막 글자가 's'인 사원을 검색하시오


			SELECT *
			FROM emp
			WHERE ename LIKE '%S';



NULL

 : 오라클에서 컬럼에 대해 저장할 값이 결정되지 않았을때 NULL을 지정합니다.
NULL도 데이터의 일종으로 정해지지 않은 값을 의미합니다.

	. 0(zero)가 아님
	. 빈 공간이 아님
	. 미확정(해당 사항 없음), 알 수 없는(unknown) 값을 의미
	. ? 혹은 무한(∞)을 의미
	. 연산, 할당, 비교가 불가능


 : 임의의 컬럼이 NULL 인 지 아닌지를 판별

	IS NULL / IS NOT NULL


. ORDER BY절
: 정렬은 크기 순서대로 나열하는 것을 말합니다.
정렬에는 2가지 방법이 존재 합니다.

	오름 차순 정렬과 내림 차순 정렬을 말합니다.


SELECT *
FROM table_name
WHERE 조건
ORDER BY 컬럼명 정렬방식


		ASC(오름차순)		DESC(내림차순)

숫자		작은 값부터 정렬	큰 값 부터 정렬
문자		사전순서대로 정렬	사전의 역순 정렬
날짜		빠른 날짜 순서로 정렬	늦은 날짜 순서로 정렬
NULL		가장 마지막에 나옴	가장 먼저 나옴


. DUAL 테이블

: DUAL 테이블이란 SYS 사용자가 소유하는 테이블이며 모든 사용자가 사용할수 있습니다.

특정한 테이블을 생성할 필요없이 dual 테이블을 이용하여 한줄의 값을 리턴 받을 수 있습니다.
